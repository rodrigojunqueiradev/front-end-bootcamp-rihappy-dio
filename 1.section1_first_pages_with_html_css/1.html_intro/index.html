Abaixo vamos falar sobre listas:

Listas ordenadas: utilizam a tag ol para a criação e li para cada item, exemplo de lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3
    1. Item 3.1
    2. Item 3.2
    3. Item 3.3

Listas não ordenadas: utilizam a tag ul para a criação e li para cada item, exemplo de lista não ordenada:

  • Item 1
  • Item 2
  • Item 3
